# Congreso Regional Norte Poniente - Módulo de Pago

Este repositorio contiene los recursos y la estructura visual de la página de confirmación de registro y métodos de pago para el Congreso Regional Norte Poniente.

## 🚀 Características del Proyecto
* **Integración con Forminator:** Gestión de captación de datos de inscritos.
* **Diseño Responsivo:** Optimizado para escritorio (PC) mediante maquetación en Divi.
* **Métodos de Pago:** Incluye visualización de datos bancarios y validación vía Código QR.
* **Flujo de Confirmación:** Instrucciones claras para el envío de comprobantes vía correo electrónico institucional.

## 🛠️ Tecnologías utilizadas
* **WordPress** como CMS.
* **Divi Builder** para la maquetación visual.
* **Forminator Pro** para la gestión de formularios.
* **ASCII Standard** para la codificación de caracteres especiales en la interfaz.

## 📝 Notas de Implementación
* El sistema utiliza el **Nombre Completo** del usuario como referencia principal para la validación de pagos en lugar de folios numéricos para simplificar la experiencia del usuario.
* El correo oficial para recepción de comprobantes es: `xxxxxxxxx`
